# Microsoft Teams Workforce Status Reporting & Automation Solution

## Project Overview
Developed a Microsoft Teams bot using Microsoft Copilot Studio (formerly Power Virtual Agents) that collects daily status updates from employees and posts summarized reports automatically in Teams channels. This project integrates Copilot Studio for the conversational bot, Power Automate for data handling and automation, SharePoint for storage, and Teams for posting.

**Key Features:**
- Greets users and collects updates (accomplishments, blockers, plans).
- Stores data in a SharePoint list.
- Summarizes updates and posts to a Teams channel.
- Low-code/no-code implementation.

**Technologies Used:**
- Microsoft Copilot Studio
- Power Automate
- SharePoint
- Microsoft Teams

**Why This Project?**
This demonstrates skills in Microsoft 365 automation, AI-driven bots, data integration, and workflow optimization. Built to streamline team reporting and reduce manual effort.

## Setup and Development Steps
For full details, see [docs/setup-guide.md](docs/setup-guide.md).

### Prerequisites
- Microsoft 365 account with access to Teams, Copilot Studio, Power Automate, and SharePoint.
- Basic Microsoft interface knowledge.

### High-Level Steps
1. Set up SharePoint site and list.
2. Create bot in Copilot Studio.
3. Integrate Power Automate flow.
4. Publish and test in Teams.
5. Enhancements for production.
